## Usage
Rename "config/config.env.env" to "config/config.env" and update the values/settings to your own

## Install dependencies
```
npm install
```

## Seed the DB
Use seeder.js to populate or empty the DB
```
# Seed the DB
node seeder -i

# Empty the DB
node seeder -d
```

## Run App
```
# Run in dev mode
npm run dev

# Run in prod mode
npm start
```
